# iOS_HW1_Calculator
Домашнее задание по курсу iOS №1 в Технопарке Mail.ru

Операции:

`+` - сложение

`-` - вычитание

`*` - умножение

`/` - деление

`mod` - остаток от деления

`!` - факториал

`^` - возведение в степень

`√` - получения квадратного корня
